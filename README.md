# Student Grade Sheet Management System (C)

This is a simple console-based Student Grade Sheet Management System written in C.

## Features
- Add student details (roll number, name, and marks)
- Automatically calculate total, percentage, and grade
- Display student grade sheets
- Menu-driven and easy to use

## Concepts Used
- Structures
- Functions
- Conditional statements
- Loops
- Basic input/output

## How to Run
1. Compile the code using a C compiler (gcc).
2. Run the executable file.
3. Follow the menu options on the screen.

## Purpose
This project is designed for beginners to understand basic student data handling in C.
